# Contribuindo com o OrgRepo

Obrigado pelo interesse em contribuir com o OrgRepo!

## Como Começar

1. Faça um fork do repositório
2. Clone o seu fork
3. Crie uma branch para sua feature/fix:
   ```bash
   git checkout -b minha-melhoria
   ```
4. Instale as dependências e rode o projeto (veja o README)

## Fluxo de Desenvolvimento Recomendado

- O estado padrão do projeto após clonar deve ser a **fundação limpa** (Resolução 44 + Campi básicos).
- Use `python manage.py load_consup44_modelos` para restaurar a fundação.
- Se precisar trabalhar com dados completos, use `load_full_data`.

## Padrões

- Siga o estilo de código existente.
- Escreva testes quando adicionar novas funcionalidades.
- Atualize a documentação quando necessário.

## Enviando sua Contribuição

1. Faça commit das suas alterações
2. Push para o seu fork
3. Abra um Pull Request

Obrigado por contribuir!
